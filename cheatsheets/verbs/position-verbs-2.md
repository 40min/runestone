# 🛋️ Står / Ligger / Sitter / Hänger — Where Is My Stuff?

---

## 🧍 Står — *Standing*

> Vertical, upright, or "in order" — everything is as it should be ✅

**Det står en golvlampa i sovrummet.** — *vertical position* (lamp stands tall 💡)
**Det står en soffa i vardagsrummet.** — *clear top/bottom side, placed properly*
**Mina skor står i hallen.** — *shoes neatly placed 👟*
**Det står i annonsen att huset är byggt 2008.** — *written in a text 📄*

➡️ Use **står** for:
| Case                                 | Example                 |
| ------------------------------------ | ----------------------- |
| 📏 Vertical position                  | lamp, bottle, person    |
| 🛋️ Object with clear top/bottom, tidy | sofa, shoes (paired up) |
| 📰 Written in a text                  | det står i tidningen... |

---

## 🛏️ Ligger — *Lying*

> Horizontal, flat, OR messy and out of place 😬

**Det ligger en gigantisk matta på golvet.** — *horizontal/flat object*
**Det ligger en massa skor i hallen.** — *same shoes as above, but now chaotic 🫣*
**Huset ligger nära en tågstation.** — *geographical location 🗺️*

➡️ Use **ligger** for:
| Case                             | Example                 |
| -------------------------------- | ----------------------- |
| 📐 Horizontal position            | rug, book lying flat    |
| 🌪️ Same object as står, but messy | shoes thrown around     |
| 🗺️ Geography / location           | staden ligger vid havet |

> 💡 **Står vs Ligger** for the same object:
> *Skorna **står** i hallen* ✅ → neat, paired
> *Skorna **ligger** i hallen* 😬 → thrown everywhere, we need to talk

---

## 📌 Sitter — *Sitting / Fixed*

> Attached, mounted, stuck — it's not going anywhere 🔩

**På väggen sitter det en högtalare.**
*There's a speaker on the wall.*

➡️ Use **sitter** when something is **fixed / attached** to a surface.
Screwed in, glued, stuck = **sitter**. 🪛

---

## 🔮 Hänger — *Hanging*

> Dangling from above, but NOT fixed — it could swing 🕯️

**Det hänger en kristallkrona i taket.**
*There's a chandelier hanging from the ceiling.*

➡️ Use **hänger** when something hangs **from top to bottom** and is **not rigidly fixed**.

---

## 🧠 Quick Cheat Card

| Verb         | Vibe           | Key trigger                             |
| ------------ | -------------- | --------------------------------------- |
| **står** 🧍   | upright & tidy | vertical / clear top-bottom / in a text |
| **ligger** 🛏️ | flat or messy  | horizontal / chaos / geography          |
| **sitter** 📌 | stuck          | fixed/attached to surface               |
| **hänger** 🔮 | dangling       | hanging from above, not fixed           |

---

## ⚡ The Shoe Test™

> Same shoes. Different verb. Different story.

|                                |                                          |
| ------------------------------ | ---------------------------------------- |
| *Skorna **står** i hallen* ✅   | Tidy, paired up. You're an adult.        |
| *Skorna **ligger** i hallen* 😤 | Thrown around. Someone's getting a talk. |
