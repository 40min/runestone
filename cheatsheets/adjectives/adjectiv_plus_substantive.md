# 🧩 Adjektiv + Substantiv — The Three Combos

---

## 🌟 The Big Picture
> There are exactly **3 combinations**. Learn these 3 and you're done. 🎯

---

## 1️⃣ Quantity words — both INDEFINITE

> en, ett, någon, något, ingen, inget, sju, många, några, inga...

|          | Adjective                            | Noun       |
| -------- | ------------------------------------ | ---------- |
| en-word  | en/någon/ingen grön **bil**          | indefinite |
| ett-word | ett/något/inget grön**t** äpple      | indefinite |
| plural   | tre/många/några/inga grön**a** bilar | indefinite |

➡️ Quantity word → adjective and noun both stay **indefinite** 📦

---

## 2️⃣ Definite article / demonstrative — both DEFINITE

> den, det, de, den här, den där, det här, det där...

|          | Adjective                          | Noun     |
| -------- | ---------------------------------- | -------- |
| en-word  | den (här/där) grön**a** bil**en**  | definite |
| ett-word | det (här/där) grön**a** äppl**et** | definite |
| plural   | de (här/där) grön**a** bil**arna** | definite |

➡️ Definite/demonstrative → adjective **-a** AND noun **definite** 🔒

---

## 3️⃣ Possessive / genitive — adjective DEFINITE, noun INDEFINITE

> min, mitt, mina, din, hans, vår + Olas, Bills, Jenny's...

|          | Adjective            | Noun                   |
| -------- | -------------------- | ---------------------- |
| en-word  | min/Olas grön**a**   | bil *(indefinite!)*    |
| ett-word | mitt/Bills grön**a** | äpple *(indefinite!)*  |
| plural   | mina/Bills grön**a** | äpplen *(indefinite!)* |

➡️ Possessive/genitive → adjective **-a** BUT noun stays **indefinite** ⚠️

---

## 🧠 The Master Cheat Table

| Combo        | Trigger words            | Adjective    | Noun     |
| ------------ | ------------------------ | ------------ | -------- |
| 1️⃣ Quantity   | en, ett, många, ingen... | obestämd     | obestämd |
| 2️⃣ Definite   | den, det, de, den här... | bestämd (-a) | bestämd  |
| 3️⃣ Possessive | min, Olas, Bills...      | bestämd (-a) | obestämd |

---

## ⚡ The Sneaky One — Combo 3

> Adjective says "I'm definite!" but noun says "I'm not!" — they disagree, and that's okay. 🤷

| ❌ Looks logical but wrong | ✅ Correct         |
| ------------------------- | ----------------- |
| min gröna bil**en**       | min gröna bil     |
| Bills gröna äppl**et**    | Bills gröna äpple |

> Possessive already signals "we know whose it is" — the noun doesn't need to be definite too. 😌
